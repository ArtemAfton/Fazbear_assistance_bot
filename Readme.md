# Fazbear assistance bot
Это бот ассистент от компании Fazbear Ent. \
Он поможет вам в ваших задачах как отличный собеседник \
**Команды бота**\
/start - запуск бота \
/help - помощь \
/debug - логи \
/tts, /stt - проверка работы speechkit \
